# Playbook: Phoenix Local Citations & NAP Submissions

**Owner:** Emily | **When to run:** Week 1 for every new client (after GBP verification, before launch)

Local ranking in Phoenix is won on citations and reviews, not on-page keywords. This playbook is the submission list we work through for every trades / local-services client.

## NAP rule (before touching any directory)
Lock **Name / Address / Phone** to whatever the GBP listing shows — exact same casing, punctuation, and suite format. Any drift between GBP, directories, and the client's website footer kills ranking. Matt's `seo-local-business.md` enforces this on the site side; we enforce it on the directory side.

## Tier 1 — do these for every client (required)
These are the citations Google's Local Pack actually trusts. Every client gets submitted here in week 1.

- [ ] **Google Business Profile** — primary. Verify first. Add services, service areas, photos (10+ original), Q&A seeded with 3–5 real answers.
- [ ] **Apple Maps** — business.apple.com → claim or add.
- [ ] **Bing Places** — bingplacesforbusiness.com.
- [ ] **Yelp** — claim existing listing if they have one (most leads do). Add services + at least 8 photos.
- [ ] **Facebook Page** — business profile with hours, services, CTA button.
- [ ] **Better Business Bureau** — bbb.org/get-accredited (paid but Phoenix trades customers check this).
- [ ] **Nextdoor Business** — nextdoor.com/business — Phoenix homeowners live here for referrals.

## Tier 2 — trades & home services
Run these when the client is HVAC, plumbing, electrical, roofing, landscaping, pool, handyman, pest, painting, concrete, fencing, flooring, garage door, junk removal, pressure washing.

- [ ] **Angi** (formerly Angie's List) — angi.com
- [ ] **HomeAdvisor** — homeadvisor.com (same parent as Angi, dual listing)
- [ ] **Thumbtack** — thumbtack.com
- [ ] **Houzz** — houzz.com (strong for remodel / flooring / fencing / landscaping)
- [ ] **Porch** — porch.com
- [ ] **Networx** — networx.com
- [ ] **Arizona ROC** — roc.az.gov license lookup (make sure client's license # is public + linked from site footer)

## Tier 3 — Phoenix-specific & local
These are the ones national citation services miss. They move the needle on "near me" and neighborhood searches.

- [ ] **Phoenix New Times Best Of** — phoenixnewtimes.com (free business listing)
- [ ] **Arizona Republic / azcentral business directory** — azcentral.com
- [ ] **Visit Phoenix** — visitphoenix.com (if consumer-facing / food / retail)
- [ ] **Phoenix Chamber of Commerce** — phoenixchamber.com (paid — recommend only for 5+ employee clients)
- [ ] **East Valley / West Valley chamber** depending on city:
  - Mesa Chamber — mesachamber.org
  - Tempe Chamber — tempechamber.org
  - Scottsdale Chamber — scottsdalechamber.com
  - Chandler Chamber — chandlerchamber.com
  - Gilbert Chamber — gilbertaz.com
  - Glendale Chamber — glendaleazchamber.org
- [ ] **Phoenix Magazine Top Doctors / Top Lawyers / Top Contractors** — if applicable
- [ ] **Arizona Foothills Magazine directory** — azfoothills.com

## Tier 4 — industry-specific (pick by vertical)
- **Restaurants / food trucks:** OpenTable, TripAdvisor, Yelp, Grubhub, DoorDash merchant, Phoenix Foodie Squad (IG-based)
- **Dentists / medical:** Healthgrades, Zocdoc, Vitals, WebMD, RateMDs
- **Auto / mobile mechanics:** CarGurus, RepairPal, MechanicAdvisor, OpenBay
- **Pet grooming / boarding:** Rover business, Wag, BringFido
- **Legal:** Avvo, Justia, FindLaw, LegalMatch, Martindale-Hubbell
- **Real estate:** Zillow Pro, Realtor.com, Homes.com
- **Fitness / gym:** ClassPass, Mindbody

## Review acceleration (run in parallel with citations)
Citations establish existence; reviews establish ranking. Aim for **10+ reviews on GBP in first 30 days** for any new client.
- [ ] Export client's last-90-day customer list (or get owner to do it)
- [ ] Draft a review-request SMS + email (Raul owns the copy)
- [ ] Set up a short review link: g.page/r/<id>/review
- [ ] Add a "Leave us a review" section to the site footer + thank-you page (Matt)
- [ ] Owner sends the requests personally, not from a marketing tool — Phoenix customers can smell automation

## NAP audit cadence
Every 90 days per active client: spot-check top 10 citations, confirm NAP still matches GBP. Fix drift immediately. Common cause: client changes phone number or moves suite and doesn't tell us.

## Tools
- **Free:** Moz Local name-search (shows inconsistencies across top directories), Whitespark citation finder (limited free tier)
- **Paid ($$ — only for 3+ active clients):** BrightLocal, Yext (avoid — sub costs compound)
- Our default: do Tier 1 + Tier 2 manually, batch Tier 3 via a shared submission spreadsheet per client.

## Timing note
Directories cache. Submit → wait 2–4 weeks for Google to crawl → re-audit. Don't re-submit within that window, you'll just create duplicate listings.
