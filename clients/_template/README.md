# Client Template

Copy this folder for each new engagement: `cp -r clients/_template clients/<client-name>`

Subfolders:
- `design/` — wireframes, mockups, brand assets, `handoff.md` for dev
- `dev/` — code or repo link, deploy notes
- `comms/` — client emails, call notes, status updates
- `content/` — copy drafts, voice/tone notes (`voice.md`)
